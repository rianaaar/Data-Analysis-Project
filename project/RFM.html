Customer Segmentation using RFM analysis¶

Overview¶

One way to retain customers is to increase customer satisfaction, by providing services following customer expectations and needs. Every customer must have different characteristics. therefore, we need to identify the characteristics of each customer and combine them in each group (customer segmentation) to make the right decisions in retaining customers.

In this project, I will use RFM to segment customers.

RFM stands for Recency, Frequency, and Monetary value, each corresponding to some key of customer behavior.

  • Recency — How recently did the customer purchase?
  • Frequency — How often do they purchase?
  • Monetary Value — How much do they spend?

Name of segments:

  • Champions : best customers, who bought most recently, most often, and spent the most.

#RFM Level name
def rfm_level(df):
    if df['rfm_score'] == '333':
        return 'Champions'
    elif ((df['rfm_score'] == '233') or (df['rfm_score'] =='323') or (df['rfm_score'] =='332')):
        return 'Loyal'
    elif ((df['rfm_score'] == '331') or (df['rfm_score'] =='322') or (df['rfm_score'] =='232') or (df['rfm_score'] =='223')):
        return 'Potential'
    elif ((df['rfm_score'] == '321') or (df['rfm_score'] =='231') or (df['rfm_score'] =='222') or (df['rfm_score'] =='213')):
        return 'Average Customer'
    elif ((df['rfm_score'] == '313') or (df['rfm_score'] =='312') or (df['rfm_score'] =='311')):
        return  'New Customer'
    elif ((df['rfm_score'] == '133') or (df['rfm_score'] =='132') or (df['rfm_score'] =='123')):
        return 'Cannot Loose Them'
    elif ((df['rfm_score'] == '111')):
        return 'Lost Customer'
    else:
        return 'Needs Attention'
RFMscores['rfm_Level'] = RFMscores.apply(rfm_level, axis=1)

Conclusion¶

We can see that 65% revenue comes from Champions, Loyal and Needs Attention. We can give business recommendations based on RFM scores.

Here's a suggested strategy for each segment:

  • Champions : Reward these customers. They will help promote our brand.
  • Loyal : Recommend related products to upsell them and help them become Champions.
  • Potential : Offer membership or recommend other products.
  • Average Customer : Build good relationships.
  • New Customer : Start building relationships with these customers.
  • Cannot Loose Them : Offer relevant promotions and helpful products.
  • Needs Attention : Send relevant promotions to reconnect. Recommend popular products/renewals at discount.
  • Lost Customer : Bring them back with relevant promotions, and run surveys to find out what went wrong.
